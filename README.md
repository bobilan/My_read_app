# My Read App

My Read App is a simple application for managing your reading list. It allows you to track books you have read, books you are currently reading, and books you plan to read. You can add new books, update their status, and perform various queries on your reading data.

## Features

- Insert new books with details such as title, description, and read status.
- Update the status, description, and progress of existing books.
- Delete books from your reading list.
- Perform queries to find out how many books you have read completely, how many books are pending, and search for books by title.

## Usage

1. Run the program and select an option from the menu.
2. Follow the prompts to perform the desired action.
3. View the results or perform additional actions as needed.
4. Exit the program when you are finished.

Enjoy managing your reading list with My Read App!
